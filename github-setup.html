GitHub Setup

Do this once. You'll create a GitHub account, then set up a simple hub page at your own web address — a home base that will link to all the projects you build from here on.

1
Sign Up for GitHub ~3 min
01
Go to github.com — GitHub is a free website where you store and publish your web apps
02
Click Sign up in the top right corner
03
Enter your email address and click Continue
04
Choose a password and a username — your username becomes part of your web address, so pick something you're happy to share publicly. Make a note of it, you'll need it in Step 2
05
Complete the short visual puzzle to confirm you're not a robot
06
Click Create account
07
Check your email — GitHub will send a confirmation link. Click it to activate your account
2
Create Your Hub Repository ~2 min
01
Once logged in, click the + icon in the top right corner and select New repository — a repository (or "repo") is a project folder on GitHub where your files live
02
Name it yourusername.github.io — swap yourusername for your actual GitHub username
⚠️ The name must match your username exactly, including upper and lower case. This special name is what tells GitHub to turn this folder into your personal website.
03
Set visibility to Public — this means anyone can visit your site, which is what we want
04
Tick ✓ Add a README file — this just creates a starter file so the folder isn't empty
05
Click Create repository
3
Add a Simple Hub Page ~5 min
01
Rather than filling this repository with a specific project, you'll put a simple hub page here — just your name and links to projects as you build them. Ask Claude to create one for you: "Create a minimal index.html homepage for a developer hub. Include my name, a short bio, and a placeholder list of projects."
02
Save the file Claude gives you as index.html on your computer
03
In your repository on GitHub, click Add file → Upload files, then drag in your index.html
04
Click Commit changes — "committing" just means saving the file to GitHub
4
Switch on Your Live Site ~5 min to go live
01
Click the Settings tab near the top of your repository page
02
In the left-hand menu, scroll down and click Pages
03
Under Source, select Deploy from a branch, set the branch to main and the folder to / (root), then click Save
💡 "Branch" and "root" are technical terms — all they mean here is: use the main folder of your project, exactly as it is. If you already see a web address on this page, you're already set up.
04
Your hub will be live at https://yourusername.github.io within a few minutes. Try refreshing if it doesn't load straight away
🎉

Your hub is live!

You now have a permanent home on the web at https://yourusername.github.io. Every project you build from here on gets its own separate repository and its own URL — you'll just add a link to this hub page each time. Head to Starting a New Project when you're ready to build your first app.
